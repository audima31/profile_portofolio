# Audima Oktasena — Portfolio

A kinetic editorial portfolio for Audima Oktasena, a frontend developer working across digital banking, web, and mobile products.

## Run locally

```bash
npm install
npm start
```

The app runs at [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm test -- --watchAll=false
npm run build
```

## Design system

- Syne for geometric display typography
- Instrument Serif for editorial contrast
- Plus Jakarta Sans for readable body copy
- Graphite, warm paper, signal orange, acid lime, and electric blue palette
- CSS-first motion with an accessible reduced-motion fallback
- Scroll reveal, progress indicator, kinetic marquee, sticky experience cards, and pointer-responsive project previews

Portfolio content is maintained in `src/data/portfolio.js`.
